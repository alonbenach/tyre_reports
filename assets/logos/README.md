# Optional Brand Logos

To render logos in PDF charts, add image files to this folder using these names:

- `pirelli.png`
- `michelin.png`
- `bridgestone.png`
- `dunlop.png`
- `continental.png`
- `metzeler.png`

`jpg`/`jpeg` are also supported.

Usage notes:
- Logos are used on the page-1 recap matrix (`RECAP BY BRAND`) and in brand strips.
- Rendering preserves original aspect ratio and centers logos in each brand box.
- If a logo is missing, the report falls back to text labels.
